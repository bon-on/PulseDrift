# PulseDrift 문서 인덱스

## 먼저 읽을 문서
- `harness/manifest.json`
- 활성 기능 명세: `harness/specs/feature-cross-platform-migration.md`
- 아키텍처 변경 전 확인 문서: `harness/adrs/adr-flutter-cross-platform-runtime.md`
- 공유 런타임 수정 전 확인 문서: `harness/contracts/contract-cross-platform-foundation.md`

## 작업 규칙
- 공통 정책은 `harness/rules/golden-rules.md`에 있다.
- 저장소 제약은 `harness/rules/constraints.md`에 있다.
- 새로운 기능 요청은 `harness/specs/` 아래 Markdown 명세로 먼저 추가한다.

## 레거시 참조
- 폐기된 Swift iOS 구현은 `legacy/swift_ios/`에 있으며, 마이그레이션 참고용으로만 사용한다.
